## Setup

##### Clone the repository

```bash
git clone https://github.com/ishandeveloper/Notes-Keeper.git
```

##### Move to the desired folder

```bash
cd \Notes-Keeper
```

##### To install the dependencies, simply write

```bash
npm install
```

##### To run the app, simply write

```bash
npm start
```
